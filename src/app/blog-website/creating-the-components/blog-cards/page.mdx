## Introduction

Blog cards are a crucial component of any blog website, providing a visually appealing way to display blog posts. They typically include the title, a brief description, and an image or thumbnail.

## Creating the Blog Card Component

To create a blog card component, we will use React and Tailwind CSS. The component will accept props for the title, description, image URL, and a link to the full post.

Since we are using Next.js with MDX, we can create a new component file in the `components` directory.

```jsx
// components/BlogCard.js
import React from "react";
import Link from "next/link";
const BlogCard = ({ title, description, imageUrl, link }) => {
  return (
    <div className="max-w-sm rounded overflow-hidden shadow-lg">
      <img className="w-full" src={imageUrl} alt={title} />
      <div className="px-6 py-4">
        <div className="font-bold text-xl mb-2">{title}</div>
        <p className="text-gray-700 text-base">{description}</p>
      </div>
      <div className="px-6 pt-4 pb-2">
        <Link href={link}>
          <a className="text-blue-500 hover:text-blue-800">Read More</a>
        </Link>
      </div>
    </div>
  );
};
export default BlogCard;
```

This component uses Tailwind CSS for styling and accepts props for the title, description, image URL, and link to the full post. The `Link` component from Next.js is used to handle navigation.

There are many ways to customize the blog card, such as adding tags, author information, or publication date. You can extend the component by adding more props and styling as needed.

### Description Shown on Hover

```jsx
import React, { useState } from "react";
import Link from "next/link";
const HoverCard = ({ title, description, imageUrl, link }) => {
  const [isHovered, setIsHovered] = useState(false);
  return (
    <div
      className="max-w-sm rounded overflow-hidden shadow-lg relative"
      onMouseEnter={() => setIsHovered(true)}
      onMouseLeave={() => setIsHovered(false)}
    >
      <img className="w-full" src={imageUrl} alt={title} />
      {isHovered && (
        <div className="absolute inset-0 bg-black bg-opacity-50 flex items-center justify-center text-white p-4">
          <p>{description}</p>
        </div>
      )}
      <div className="px-6 py-4">
        <div className="font-bold text-xl mb-2">{title}</div>
        <Link href={link}>
          <a className="text-blue-500 hover:text-blue-800">Read More</a>
        </Link>
      </div>
    </div>
  );
};
export default HoverCard;
```
