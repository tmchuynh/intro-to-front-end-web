# Individual Blog Post Pages

## Table of Contents

## Introduction

This page is dedicated to displaying individual blog posts. Each post will have its own unique content from the MDX filed located in the `content/posts` directory of your project.

## Content

Here, we will be using `generateStaticParams` to generate static paths for each blog post based on the filenames in the `content/posts` directory. This allows Next.js to pre-render each blog post page at build time.

```tsx
export default async function Page({
  params,
}: {
  params: Promise<{ slug: string }>;
}) {
  const { slug } = await params;
  const { default: Post } = await import(`@/content/${slug}.mdx`);

  return <Post />;
}

export function generateStaticParams() {
  return [{ slug: "welcome" }, { slug: "about" }];
}

export const dynamicParams = false;
```

This code dynamically imports the MDX file corresponding to the `slug` parameter and renders it as a React component. The `generateStaticParams` function generates static paths for the blog posts, allowing Next.js to pre-render them.

Now you are ready to create individual blog post pages in your Next.js application. Each post will be accessible via its unique slug, and the content will be rendered from the corresponding MDX file.

Let's start by creating a simple blog post in the `content/posts` directory. Head over to [Writing the Blog Posts](/blog-website/writing-the-blog-posts) to learn how to create and format your blog posts using MDX.

## About `generateStaticParams`

The `generateStaticParams` function is used to define the static paths that Next.js will pre-render at build time. In this example, it returns an array of objects, each containing a `slug` property that corresponds to the filenames of the MDX files in the `content/posts` directory.

### Parameters

- `params`: This is a promise that resolves to an object containing the `slug` of the blog post.
- `slug`: The slug is derived from the filename of the MDX file, which is used to dynamically import the content of the blog post.

### Return Value

- The function returns an array of objects, each with a `slug` property that Next.js uses to generate static paths for the blog posts. This allows the application to pre-render each blog post page at build time, improving performance and SEO.

### Example Routes

| Route                                  | `generateStaticParams` Return Type                     |
| -------------------------------------- | ------------------------------------------------------ |
| `/blog/welcome`                        | `{ slug: "welcome" }`                                  |
| `/product/[id]`                        | `{ id: "1" }`                                          |
| `/categories/[category]/[subCategory]` | `{ category: "books", subCategory: "fiction" }`        |
| `/posts/[year]/[month]/[slug]`         | `{ year: "2023", month: "10", slug: "my-first-post" }` |
| `/products/[...slug]`                  | `{ slug: ["electronics", "laptop"] }`                  |

You can define more complex routes by using nested parameters or catch-all routes. The `generateStaticParams` function will return the appropriate parameters for each route, allowing Next.js to pre-render them at build time.

### Example Pages for Each Route

#### `/blog/welcome`

```tsx
import Welcome from "@/markdown/welcome.mdx";

export default function Page() {
  return <Welcome />;
}
```

#### `/product/[id]`

```tsx
export default async function Page({
  params,
}: {
  params: Promise<{ id: string }>;
}) {
  const { id } = await params;
  const { default: Post } = await import(`@/content/${id}.mdx`);

  return <Post />;
}

export function generateStaticParams() {
  return [{ id: "welcome" }, { id: "about" }];
}

export const dynamicParams = false;
```

#### `/categories/[category]/[subCategory]`

```tsx
export default async function Page({
  params,
}: {
  params: Promise<{ category: string; subCategory: string }>;
}) {
  const { category, subCategory } = await params;
  const { default: Post } = await import(
    `@/content/${category}/${subCategory}.mdx`
  );
  return <Post />;
}
export function generateStaticParams() {
  return [
    { category: "books", subCategory: "fiction" },
    { category: "electronics", subCategory: "laptops" },
  ];
}
export const dynamicParams = false;
```

#### `/posts/[year]/[month]/[slug]`

```tsx
export default async function Page({
  params,
}: {
  params: Promise<{ year: string; month: string; slug: string }>;
}) {
  const { year, month, slug } = await params;
  const { default: Post } = await import(
    `@/content/posts/${year}/${month}/${slug}.mdx`
  );
  return <Post />;
}
export function generateStaticParams() {
  return [
    { year: "2023", month: "10", slug: "my-first-post" },
    { year: "2023", month: "11", slug: "my-second-post" },
  ];
}
export const dynamicParams = false;
```

#### `/products/[...slug]`

```tsx
export default async function Page({
  params,
}: {
  params: Promise<{ slug: string[] }>;
}) {
  const { slug } = await params;
  const { default: Post } = await import(
    `@/content/products/${slug.join("/")}.mdx`
  );
  return <Post />;
}
export function generateStaticParams() {
  return [{ slug: ["electronics", "laptop"] }, { slug: ["books", "fiction"] }];
}
export const dynamicParams = false;
```
