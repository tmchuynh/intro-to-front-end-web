# CLI Plugins and Add-ons

Command Line Interfaces (CLIs) can be enhanced with various plugins and add-ons to improve functionality, usability, and productivity. These plugins can provide features like syntax highlighting, command suggestions, custom themes, and more. Below is a list of popular plugins and add-ons for different operating systems, including macOS, Linux, and Windows.

## Table of Contents

## macOS Terminal

- [iTerm2](https://iterm2.com/): A popular terminal emulator for macOS with extensive features.
- [Oh My Zsh](https://ohmyz.sh/): A framework for managing Zsh configurations, providing themes and plugins.
- [Hyper](https://hyper.is/): A terminal built on web technologies, allowing for customization with plugins and themes.
- [Alacritty](https://alacritty.org/): A fast, GPU-accelerated terminal emulator that supports custom configurations.
- [Fish Shell](https://fishshell.com/): A user-friendly shell with features like syntax highlighting and autosuggestions.

## Linux Shell

- [Oh My Zsh](https://ohmyz.sh/): A framework for managing Zsh configurations, providing themes and plugins.
- [Fish Shell](https://fishshell.com/): A user-friendly shell with features like syntax highlighting and autosuggestions.
- [Bash-it](https://github.com/Bash-it/bash-it): A collection of community-driven Bash commands and scripts.
- [Powerlevel10k](https://github.com/romkatv/powerlevel10k): A fast and highly customizable Zsh theme that enhances the command prompt.
- [Tilda](https://github.com/lanoxx/tilda): A drop-down terminal emulator that can be used with various shells.
- [Terminator](https://gnome-terminator.org/): A terminal emulator that allows for multiple terminal windows in a single window, with features like tabs and split views.
- [Tilix](https://gnunn1.github.io/tilix-web/): A tiling terminal emulator that allows users to split the terminal into multiple panes and tabs, enhancing multitasking capabilities.
- [Termux](https://termux.com/): A terminal emulator for Android that provides a Linux environment with access to a package manager, allowing users to install and run Linux applications on their Android devices.
- [Zsh Autosuggestions](https://github.com/zsh-users/zsh-autosuggestions): A Zsh plugin that suggests commands as you type, based on your command history and completions.
- [Zsh Syntax Highlighting](https://github.com/zsh-users/zsh-syntax-highlighting): A Zsh plugin that provides syntax highlighting for commands as you type, making it easier to spot errors.
- [fzf](https://github.com/junegunn/fzf): A command-line fuzzy finder that can be integrated with various shells to enhance command history and file searching.
- [tmux](https://github.com/tmux/tmux): A terminal multiplexer that allows users to manage multiple terminal sessions within a single window, providing features like session persistence and window splitting.

## Windows Command Processor

- [Cmder](https://cmder.net/): A portable console emulator for Windows that provides a better command-line experience with features like tabs and Git integration.
- [ConEmu](https://conemu.github.io/): A customizable terminal emulator for Windows that supports multiple console applications and provides features like tabs, split views, and customizations.
- [Windows Terminal](https://aka.ms/terminal): A modern terminal application for Windows that supports multiple tabs, Unicode, and customizations. It can run PowerShell, Command Prompt, and WSL (Windows Subsystem for Linux) sessions.
- [PowerShell](https://docs.microsoft.com/en-us/powershell/): A task automation and configuration management framework that includes a command-line shell and scripting language. It provides advanced features for system administration and automation tasks.
- [Git Bash](https://gitforwindows.org/): A terminal emulator that provides a Bash shell on Windows, allowing users to run Unix-like commands and scripts. It is commonly used for Git version control operations.
- [Windows Subsystem for Linux (WSL)](https://docs.microsoft.com/en-us/windows/wsl/): A compatibility layer that allows users to run Linux distributions directly on Windows. It provides access to Linux command-line tools and utilities, enabling developers to use familiar Linux commands within the Windows environment.
- [Chocolatey](https://chocolatey.org/): A package manager for Windows that allows users to install, update, and manage software packages from the command line. It simplifies software installation and management by providing a command-line interface for package management.
- [PowerToys](https://github.com/microsoft/PowerToys): A set of utilities for power users to enhance the Windows experience. It includes features like a file explorer extension, window manager, and keyboard shortcuts, providing additional functionality to the Windows command line.
- [Windows Terminal Preview](https://aka.ms/terminal-preview): A preview version of the Windows Terminal that includes experimental features and improvements. It allows users to test new functionalities before they are released in the stable version.
