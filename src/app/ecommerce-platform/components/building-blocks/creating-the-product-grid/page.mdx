# The Product Grid

## Table of Contents

## Introduction

The Product Grid is a key component of the e-commerce platform that displays a grid of products available for purchase. It allows users to browse through products, view details, and add items to their cart. The grid layout provides an organized and visually appealing way to showcase products, making it easier for users to find what they are looking for.

## The `import` Statements


##### Explanation of the Code

Here, we are using our `ProductItem` interface to define the structure of the products that will be displayed in the grid. The `useAuth`, `useCart`, `useCurrency`, and `useWishlist` hooks are imported to manage user authentication, cart operations, currency formatting, and wishlist functionality, respectively. The `Badge`, `Button`, `Card`, and `CardContent` components from our UI library are used to create a visually appealing product card layout. The `displayRatingStars` function is used to render product ratings, and the `cn` utility function helps with conditional class names.

The `viewMode` prop allows us to switch between grid and list views, providing flexibility in how products are displayed.

A little reminder, our `ProductItem` interface is defined in the `src/lib/interfaces/product.ts` file and looks like this:

## The `ProductGrid` Component

```tsx
// src/components/products/ProductGrid.tsx

export default function ProductGrid({
  products,
  viewMode = "grid",
}: ProductGridProps) {
  const { formatPrice } = useCurrency();
  const { addToCart, isInCart } = useCart();
  const { addToWishlist, removeFromWishlist, isInWishlist } = useWishlist();
  const { isLoggedIn } = useAuth();

  const toggleWishlist = async (product: ProductItem, e: React.MouseEvent) => {
    e.preventDefault();
    e.stopPropagation();

    if (!isLoggedIn) {
      toast.error("Please log in to add items to your wishlist");
      return;
    }

    if (isInWishlist(product.id)) {
      const result = await removeFromWishlist(product.id);
      if (result.success) {
        toast.success(result.message);
      } else {
        toast.error(result.message);
      }
    } else {
      const result = await addToWishlist(product);
      if (result.success) {
        toast.success(result.message);
      } else {
        toast.error(result.message);
      }
    }
  };

  const handleAddToCart = (product: ProductItem, e: React.MouseEvent) => {
    e.preventDefault();
    e.stopPropagation();

    const wasInCart = isInCart(product.id);
    addToCart(product);

    if (wasInCart) {
      toast.success(`Added another ${product.title} to cart!`);
    } else {
      toast.success(`${product.title} added to cart!`);
    }
  };

```
