## Handling Discounts

```tsx startLineNumber=70
// src/app/cart/page.tsx
const handleApplyDiscount = async () => {
  if (!discountCode.trim()) {
    toast.error("Please enter a discount code");
    return;
  }

  setIsApplyingDiscount(true);
  const result = applyDiscount(discountCode.trim().toUpperCase());

  if (result.success) {
    toast.success(result.message);
    setDiscountCode("");
  } else {
    toast.error(result.message);
  }
  setIsApplyingDiscount(false);
};

const handleRemoveDiscount = () => {
  removeDiscount();
  toast.info("Discount removed");
};
```

##### Explanation of the Code

This code handles the application and removal of discount codes. It provides feedback to the user through toast notifications and manages the loading state during the discount application process.
