## Base Content

When you create a new Next.js project, it comes with a default `globals.css` file. This file is located in the `src/app/globals.css` directory. You can use this file to define global styles for your application.

##### NOTE

> This is not the entire `globals.css` file, but only the relevant parts for this section.

### Theme Variables

The `globals.css` file also defines a set of CSS variables that are used throughout the application. These variables are used to define the colors and styles for various elements in the application. You can customize these variables to change the look and feel of your application.

You can find these variables in the `:root` selector at the top of the `globals.css` file. These variables are used to define the colors and styles for various elements in the application. You can customize these variables to change the look and feel of your application.

## Using `@layer` to Apply Styles

The `@layer` rule in CSS allows you to organize your styles into layers, which can help with specificity and maintainability. You can define different layers for different types of styles, such as base styles, component styles, and utility styles.

```css
/* src/app/globals.css */

@layer components {
  Card,
  AlertDialog,
  Dialog {
    @apply bg-card text-card-foreground border-border rounded-lg shadow-md;
  }
  Popover {
    @apply bg-popover text-popover-foreground border-border rounded-lg shadow-md;
  }
  Image {
    @apply object-cover object-center rounded-2xl;
  }
}

@layer base {
  * {
    @apply border-border outline-ring/50;
  }
  body {
    @apply bg-background font-[NotoSans] text-foreground cursor-default;
  }

  h1 {
    @apply font-[Gloock] mb-2 font-bold text-4xl;
  }

  h2 {
    @apply font-[EncodeSans] font-extrabold mb-3 text-2xl lg:text-3xl;
  }

  h3 {
    @apply font-[NotoSans] font-bold text-base tracking-wider lg:text-xl;
  }

  h4 {
    @apply font-[Gloock] lg:text-lg text-primary underline text-base;
  }
  h5 {
    @apply font-[Gloock] uppercase font-medium text-gray-600 dark:text-gray-400 text-xs;
  }
  p {
    @apply text-pretty leading-relaxed;
  }
  blockquote {
    @apply mt-2 italic mb-5 text-lg max-w-4xl;
  }
  header {
    @apply mb-8 w-full;
  }

  header h1,
  header h5,
  header blockquote {
    @apply text-start;
  }

  section {
    @apply mb-10;
  }

  strong {
    @apply font-semibold;
  }
}

@layer base {
  * {
    @apply border-border outline-ring/50;
  }
  body {
    @apply bg-background text-foreground;
  }
}
```
