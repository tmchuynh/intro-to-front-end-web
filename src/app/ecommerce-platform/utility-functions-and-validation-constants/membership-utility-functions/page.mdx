# Membership Utility Functions

## Table of Contents

## The `import` Statements

```tsx
// src/lib/utils/membership.tsx
import { Crown, Star, User, Zap } from "lucide-react";
```

##### Explanation of the Code

The `import` statements bring in the necessary icons from the `lucide-react` library, which will be used to visually represent different membership statuses and features within the e-commerce platform.

## The Membership Tier Icons

```tsx
// src/lib/utils/membership.tsx
export const getTierIcon = (tierName?: string) => {
  switch (tierName?.toLowerCase()) {
    case "basic":
      return <Star className="w-5 h-5" />;
    case "premium":
      return <Zap className="w-5 h-5" />;
    case "vip":
      return <Crown className="w-5 h-5" />;
    default:
      return <User className="w-5 h-5" />;
  }
};
```

##### Explanation of the Code

This function returns the appropriate icon based on the membership tier name. It uses a switch statement to determine which icon to return, providing a visual representation of the membership level.

## The Membership Status Colors

```tsx
// src/lib/utils/membership.tsx
export const getTierBadgeVariant = (tierName: string) => {
  switch (tierName.toLowerCase()) {
    case "basic":
      return "gradient";
    case "premium":
      return "platinum";
    case "vip":
      return "premium";
    default:
      return "outline";
  }
};
```

##### Explanation of the Code

This function determines the badge variant based on the membership tier name. It returns a specific style for each tier, ensuring that the visual representation aligns with the membership level.

## The Membership Tier Colors

```tsx
// src/lib/utils/membership.tsx
export const getTierColor = (tierName?: string) => {
  switch (tierName?.toLowerCase()) {
    case "basic":
      return "bg-blue-500";
    case "premium":
      return "bg-purple-500";
    case "vip":
      return "bg-yellow-500";
    default:
      return "bg-gray-500";
  }
};
```

##### Explanation of the Code

This function returns the appropriate background color for the membership tier. It uses a switch statement to determine which color to return, providing a consistent visual cue for each membership level.
