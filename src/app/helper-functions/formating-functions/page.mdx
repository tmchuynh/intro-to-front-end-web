# Formatting Functions

## Table of Contents

## Format Number to Currency

```javascript
function formatToCurrency(num) {
  return new Intl.NumberFormat("en-US", {
    style: "currency",
    currency: "USD",
  }).format(num);
}
```

**Use Case**: This function is useful for displaying numbers as currency, making it easier for users to read and understand financial values.

## Format Date to Readable String

```javascript
function formatDate(date) {
  return new Intl.DateTimeFormat("en-US", {
    year: "numeric",
    month: "long",
    day: "numeric",
  }).format(new Date(date));
}
```

**Use Case**: This function is useful for converting date strings into a more human-readable format, such as "January 1, 2023", which is easier for users to understand.

## Format Phone Number

```javascript
function formatPhoneNumber(phone) {
  return phone
    .replace(/\D/g, "") // Remove non-digit characters
    .replace(/(\d{3})(\d{3})(\d{4})/, "($1) $2-$3");
}
```

**Use Case**: This function formats a phone number string into a standard format, such as "(123) 456-7890", making it easier for users to read and input phone numbers.

## Changing String Formats

### Format to Title Case

```javascript
function toTitleCase(str) {
  return str
    .toLowerCase()
    .split(" ")
    .map((word) => word.charAt(0).toUpperCase() + word.slice(1))
    .join(" ");
}
```

**Use Case**: This function is useful when you want to format a string so that each word starts with an uppercase letter, such as in titles or headings.

### Format to CamelCase

```javascript
function toCamelCase(str) {
  return str
    .toLowerCase()
    .split(" ")
    .map((word, index) =>
      index === 0 ? word : word.charAt(0).toUpperCase() + word.slice(1),
    )
    .join("");
}
```

**Use Case**: This function is commonly used in programming to format identifiers, such as variable names or function names, where the first word is lowercase and subsequent words are capitalized.

### Format to Snake Case

```javascript
function toSnakeCase(str) {
  return str.toLowerCase().split(" ").join("_");
}
```

**Use Case**: This function is useful for formatting strings in a way that is often used in programming languages for variable names, where words are separated by underscores.

### Format to Kebab Case

```javascript
function toKebabCase(str) {
  return str.toLowerCase().split(" ").join("-");
}
```

**Use Case**: This function is commonly used in URLs or CSS class names, where words are separated by hyphens.
