# Basic Data Generation

## Table of Contents

## Generate Random Numbers

```javascript
function generateRandomNumber(min, max) {
  return Math.floor(Math.random() * (max - min + 1)) + min;
}
```

**Use Case**: This function generates a random integer between the specified minimum and maximum values, which can be useful for creating random data in tests or simulations.

## Generate Random Strings

```javascript
function generateRandomString(length) {
  const characters =
    "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789";
  let result = "";
  for (let i = 0; i < length; i++) {
    result += characters.charAt(Math.floor(Math.random() * characters.length));
  }
  return result;
}
```

**Use Case**: This function generates a random alphanumeric string of a specified length, which can be useful for creating unique identifiers, passwords, or test data.

## Generate Random Dates

```javascript
function generateRandomDate(start, end) {
  const startDate = new Date(start);
  const endDate = new Date(end);
  const randomTime =
    startDate.getTime() +
    Math.random() * (endDate.getTime() - startDate.getTime());
  return new Date(randomTime);
}
```

**Use Case**: This function generates a random date between two specified dates, which can be useful for creating test data or simulating events in applications.

## Generate Random Boolean

```javascript
function generateRandomBoolean() {
  return Math.random() < 0.5;
}
```

**Use Case**: This function generates a random boolean value (true or false), which can be useful for testing conditions or simulating user choices in applications.

## Generate Random Array

```javascript
function generateRandomArray(length, min, max) {
  return Array.from({ length }, () => generateRandomNumber(min, max));
}
```

**Use Case**: This function generates an array of random integers of a specified length, with values between the specified minimum and maximum. It can be useful for creating test data or simulating datasets in applications.
