# Loading Component

## Table of Contents

## About This Component

This component is used to display a loading state in the portfolio website. It is typically used when fetching data or performing asynchronous operations.

## Usage

You can use this loading component in your pages to indicate that content is being loaded. Below is an example of how to implement a simple loading component using React.

```tsx
import React, { useState } from "react";
import Loading from "@/components/Loading";
function MyComponent() {
  const [loading, setLoading] = useState(true);

  if (loading) {
    return <Loading />;
  }
  return (
    <div>
      {/* Your main content goes here */}
      <h1>Welcome to My Portfolio</h1>
      <p>This is the main content of the page.</p>
    </div>
  );
}
```

### Progress Bar Loading

A loading component with a progress bar for longer operations.

```tsx
import React, { useState, useEffect } from "react";

const ProgressBarLoading: React.FC = () => {
  const [progress, setProgress] = useState(0);

  useEffect(() => {
    const timer = setInterval(() => {
      setProgress((prevProgress) => {
        if (prevProgress >= 100) {
          clearInterval(timer);
          return 100;
        }
        return prevProgress + 10;
      });
    }, 200);

    return () => clearInterval(timer);
  }, []);

  return (
    <div className="flex items-center justify-center min-h-screen bg-slate-900 text-white">
      <div className="w-80 space-y-4">
        <h2 className="text-xl font-semibold text-center">Loading Portfolio</h2>
        <div className="w-full bg-gray-700 rounded-full h-2">
          <div
            className="bg-blue-500 h-2 rounded-full transition-all duration-300"
            style={{ width: `${progress}%` }}
          ></div>
        </div>
        <p className="text-center text-gray-300">{progress}% Complete</p>
      </div>
    </div>
  );
};
export default ProgressBarLoading;
```

##### Customization Options

- **Progress speed**: Change the interval timing from `200` milliseconds to `100` (faster) or `500` (slower)
- **Increment size**: Modify `+ 10` to `+ 5` (slower progress) or `+ 20` (faster progress)
- **Bar width**: Adjust container width from `w-80` to `w-96` (wider) or `w-64` (narrower)
- **Bar height**: Change `h-2` to `h-1` (thinner) or `h-4` (thicker)
- **Colors**: Replace `bg-blue-500` with your brand colors (e.g., `bg-green-500`, `bg-purple-500`)
- **Background theme**: Change `bg-slate-900` to `bg-white` and adjust text colors accordingly
- **Loading text**: Customize the title and percentage text to match your application
- **Animation**: Modify `duration-300` for different transition speeds
- **Real progress**: Connect to actual loading progress instead of simulated progress

### Framer Motion Loading

Advanced loading animation using Framer Motion for smooth transitions.

```tsx
import React from "react";
import { motion } from "framer-motion";

const FramerMotionLoading: React.FC = () => {
  const containerVariants = {
    start: {
      transition: {
        staggerChildren: 0.2,
      },
    },
    end: {
      transition: {
        staggerChildren: 0.2,
      },
    },
  };

  const circleVariants = {
    start: {
      y: "0%",
    },
    end: {
      y: "100%",
    },
  };

  const circleTransition = {
    duration: 0.5,
    repeat: Infinity,
    repeatType: "reverse" as const,
    ease: "easeInOut",
  };

  return (
    <div className="flex items-center justify-center min-h-screen bg-black">
      <div className="flex flex-col items-center space-y-8">
        <motion.div
          className="flex space-x-2"
          variants={containerVariants}
          initial="start"
          animate="end"
        >
          {[1, 2, 3, 4, 5].map((circle) => (
            <motion.div
              key={circle}
              className="w-4 h-4 bg-white rounded-full"
              variants={circleVariants}
              transition={circleTransition}
            />
          ))}
        </motion.div>
        <motion.p
          className="text-white text-xl font-light"
          initial={{ opacity: 0 }}
          animate={{ opacity: 1 }}
          transition={{ delay: 0.5 }}
        >
          Crafting your experience...
        </motion.p>
      </div>
    </div>
  );
};
export default FramerMotionLoading;
```

##### Customization Options

- **Circle count**: Change `[1, 2, 3, 4, 5]` to fewer or more circles
- **Stagger timing**: Modify `staggerChildren: 0.2` to `0.1` (faster) or `0.3` (slower)
- **Animation duration**: Change `duration: 0.5` to adjust how fast circles move
- **Movement distance**: Replace `y: "100%"` with `y: "50%"` or `y: "150%"`
- **Circle colors**: Change `bg-white` to any color or create gradient circles
- **Background**: Replace `bg-black` with other backgrounds or gradients
- **Animation type**: Change `y` movement to `x`, `scale`, or `rotate` for different effects
- **Easing**: Modify `ease: "easeInOut"` to `"easeOut"`, `"easeIn"`, or custom cubic-bezier
- **Text animation**: Add more complex text animations with `variants`
- **Entry animations**: Add `initial` and `animate` props for page entry effects
- **Circle size**: Adjust `w-4 h-4` for different circle sizes

## Implementation Tips

### Choosing the Right Loading Component

- **Simple spinners**: Best for quick operations (< 3 seconds)
- **Progress bars**: Ideal for operations with measurable progress
- **Skeleton loaders**: Perfect for content that has a predictable layout
- **Branded loaders**: Great for initial app loading or page transitions

### Performance Considerations

```tsx
// Lazy load the loading component to reduce bundle size
const LazyLoading = React.lazy(() => import("@/components/Loading"));

// Use with Suspense for better UX
function App() {
  return (
    <Suspense fallback={<div>Loading...</div>}>
      <LazyLoading />
    </Suspense>
  );
}
```
