# Loading Component

## Table of Contents

## About This Component

This component is used to display a loading state in the portfolio website. It is typically used when fetching data or performing asynchronous operations.

## Usage

You can use this loading component in your pages to indicate that content is being loaded. Below is an example of how to implement a simple loading component using React.

```tsx
import React, { useState } from "react";
import Loading from "@/components/Loading";
function MyComponent() {
  const [loading, setLoading] = useState(true);

  if (loading) {
    return <Loading />;
  }
  return (
    <div>
      {/* Your main content goes here */}
      <h1>Welcome to My Portfolio</h1>
      <p>This is the main content of the page.</p>
    </div>
  );
}
```

## Using Chakra UI Spinner

You can use the Chakra UI Spinner component to create a loading indicator. Below is an example of how to implement a simple loading component using Chakra UI.

```tsx
import React from "react";
import { Spinner } from "@chakra-ui/react";
const Loading: React.FC = () => {
  return (
    <div
      style={{
        display: "flex",
        justifyContent: "center",
        alignItems: "center",
        height: "100vh",
      }}
    >
      <Spinner size="xl" />
    </div>
  );
};
export default Loading;
```
