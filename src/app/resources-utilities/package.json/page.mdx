## Helpful Scripts
```json
{
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint",
    "format": "prettier --write .",
    "test": "jest"
  }
}
```

## Use Cases
- **`dev`**: Starts the Next.js development server for local development.
- **`build`**: Builds the application for production, optimizing performance and generating static files.
- **`start`**: Starts the production server after building the application.
- **`lint`**: Runs ESLint to check for code quality and style issues.
- **`format`**: Formats the codebase using Prettier, ensuring consistent code style.
- **`test`**: Runs tests using Jest, ensuring code functionality and reliability.
